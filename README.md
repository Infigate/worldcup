# ワールドカップ2026 観戦ガイド

FIFAワールドカップ2026(アメリカ・カナダ・メキシコ大会)の観戦Webアプリ。Vite + React + TypeScript製。

## 機能

- **グループステージ** — 12組(A〜L)の順位表と試合結果。実際の組分け・日程に基づく
- **決勝トーナメント** — ラウンド32〜決勝のブラケット可視化。各組の順位確定にともない対戦カードが自動で埋まる
- **日程・放送** — 全104試合を日本時間/現地時刻併記で一覧。視聴ツール(DAZN・NHK BSプレミアム4K・地上波)をチップ表示
- **マイフォロー** — 国・注目選手を選択して関連試合を追跡(選択はlocalStorageに保存)

## 起動

```bash
npm install
npm run dev
```

## 試合結果の更新

結果は [src/data/results.json](src/data/results.json) で管理し、アプリはこのファイルの内容を表示します。

### 自動更新(推奨)

```bash
npm run update-results
```

[openfootball/worldcup.json](https://github.com/openfootball/worldcup.json)(パブリックドメイン)からスコアを取得して反映します。フィード側に結果が未反映の場合は何も更新されません。

### 手動更新

`src/data/results.json` の `results` に試合番号で追記します:

```jsonc
{
  "results": {
    "m1": { "score": [2, 0] },                          // グループステージ
    "m90": { "score": [1, 1], "pk": [4, 2] },           // ノックアウト(PK決着)
    "m92": { "score": [2, 1], "et": true }              // 延長決着
  },
  "slots": {
    "m74": { "away": "mar" }                            // 3位通過チームの振り分け(手動指定)
  }
}
```

- 試合番号はFIFA公式のMatch 1〜104に対応(`m1`〜`m104`)
- ノックアウトの勝者は自動で次ラウンドに進みます
- R32の「組1位/2位」枠は順位表から自動解決されます。**3位通過チームの振り分けのみ**FIFA規定の組み合わせ表によるため、`slots` で手動指定してください

## データソース

- 組分け・日程・会場: FIFA公式発表 / Wikipedia (2026-06-12取得)
- 放送・配信: 2026年6月時点の各社発表(DAZN全104試合、NHK BSプレミアム4K全試合、日本戦の地上波はNHK総合・日本テレビ)。変更される場合があります
